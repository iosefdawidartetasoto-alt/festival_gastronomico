# Fusion Gourmet Festival — Landing Page

## Description
A responsive landing page for a gastronomic festival, built as a performance test for HTML & CSS skills. Features a dark, elegant design with gold accents, including a hero section, schedule table, about section, chef cards, multimedia grid, and footer.

## Technologies Used
- HTML5 (semantic elements: `header`, `main`, `section`, `article`, `footer`, `nav`)
- CSS3 (Flexbox, Grid, custom properties, media queries, transitions)

## Project Structure
```
/
├── landing.html      # Main HTML file
├── style.css         # Stylesheet
├── logo.JPG          # Festival logo
└── platillo.JPG      # Hero background image
```

## How to View Locally
1. Clone or download this repository.
2. Make sure all files (`landing.html`, `style.css`, `logo.JPG`, `platillo.JPG`) are in the same folder.
3. Open `landing.html` in any modern web browser (Chrome, Firefox, Edge, Safari).
4. No server or dependencies required — it runs entirely in the browser.
